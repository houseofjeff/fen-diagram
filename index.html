Welcome to fen-diagram

This is a simple-to-use chess diagram library, using the Forsyth-Edwards Notation to define any possible (or impossible) chess board layout. For instance, this board layout...


...is represented by this FEN string:
4r1k1/1b3p1p/ppq3p1/2p5/8/1P3R1Q/PBP3PP/7K.

Easy-to-use

Include the library:
<script src="fendiagram.js"></script>

Add a div to hold the board:
<div id="board1"></board>

Create the board and include some FEN:
<script>
  ChessBoard("board1","4r1k1/1b3p1p/ppq3p1/2p5/8/1P3R1Q/PBP3PP/7K");
</script>


No prerequisites

No jquery, no angular, no nothing. Just a little javascript.
